# Тестер простых matplotlib-программ в ejudge

Программа предназначена для тестирования простых программ построения диаграмм при помощи библиотеки `matplotlib`.

По идеям [https://github.com/ShashkovS/matplotlib_tester](https://github.com/ShashkovS/matplotlib_tester)  

Программа пользователя переделывается так, что вместо диаграммы на вывод попадает ее текстовое описание, которое легко анализировать. Часто подходит простое сравнение выводов пользовательской и эталонной программы.

## Основные отличия

Программа не использует саму библиотеку `matplotlib`. Все вызовы ее команд перехватываются и результат выводится как текст. 

## Использование

Тестер является оберткой пользовательской программы, подключается через настройку задачи `start_cmd`:

```
start_cmd = "../matplotlib_tester.py"
```

Тестер настраивается на различные проверки через переменные окружения, поэтому его можно положить в директорию `problems` и сделать общим для всех задач контеста.

### Настройки

Через переменные окружения тестер настраивается на вывод только нужных параметров диаграммы.

В настройках задачи нужно добавить переменные `start_env`, например, настройка

```
start_env = "PRINT_LINES=1"
start_env = "PRINT_BARS=1"
```
добавит в вывод информацию о линиях и столбиках (только координаты).

**Полный список настроек**

**PRINT_LINES** — 0/1 *(печать линий и их точек из `plt.plot`)*  
**PRINT_BARS** — 0/1 *(печать столбцов из `plt.bar`)*  
**PRINT_PIES** — 0/1 *(печать секторов из `plt.pie` — углы, радиус)*  
**PRINT_PIE_LABELS** — 0/1 *(печать подписей секторов pie и их координат)*  
**PRINT_XTICKS** — 0/1 *(печать подписей по оси X)*  
**PRINT_TITLE** — 0/1 *(печать заголовка графика)*  
**PRINT_AXIS** — 0/1 *(печать подписей осей X и Y)*  
**PRINT_LABELS** — 0/1 *(печать `label=` у линий)*  
**PIE_LABELS** — строка `"A,B,C"` *(список обязательных подписей для проверки в pie)*  

### Пример программы и вывода тестера 

Программа
```
import matplotlib.pyplot as plt

values = [40, 30, 20, 10]
labels = ['Python', 'C++', 'Java', 'Other']

plt.pie(values, labels=labels)
plt.show()
```
Настройка
```
start_env = "PRINT_PIES=1"
start_env = "PRINT_PIE_LABELS=1"
```

### Ограничения пользовательской программы

Программа должна импортировать `pyplot` с именем `plt` и рисовать на нем.
```python
import matplotlib.pyplot as plt
...
plt.plot(...)
plt.bar(...)
plt.pie(...)
...
plt.show()
```
